## Quick Start

### Development

```bash
# Install dependencies
npm install
```

```bash
# Start development server
npm run dev
```
### Production

Using Docker:

```bash
# Build and run with docker-compose
docker-compose up -d
 ```

Or traditional deployment:

```bash
# Build for production
npm run build

# Preview production build
npm run preview
 ```
